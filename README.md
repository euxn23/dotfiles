# Dotfiles
## Tools
- editorconfig
- git(including tig)
- neovim
- bash
- tmux 2.2+
- vim 7.4+
- zsh 4.3.11+

## Usage
- `$ git clone https://github.com/euxn23/dotfiles.git`
- Makefile provides commands to create symbolic links.
    - simply `$ make` to setup all tools
    - or specify make command as `$ make <tool>` to setup one.
        - e.g. `$ make zsh`
