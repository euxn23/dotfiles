# Dotfiles
## Tools
- editorconfig
- git(including tig)
- neovim
- profile
- tmux
- vim
- zsh

## Usage
- `$ git clone https://github.com/yutaszk/dotfiles.git ~/.dotfiles`
- Makefile provides commands to createsymbolic links.

### Single Activation
- `$ make <tool>`
    - e.g. `$ make zsh`
### All Activation
- `$ make`
